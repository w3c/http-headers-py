CGI interface to tidy

a branch (from v 1.11 2002/08/02 13:40:38 dom) of the implementation of the tidy service running at w3.org since Apr 2000.

By popular demand, also a branch (from headers,v 1.24 2002/10/18 16:07:51 dom) of the HTTP Head CGI service running at W3.org, and a branch (from spellchecker,v 1.46 2005/01/10 09:33:47 dom) of the spellchecker service.

tidy and spellchecker need http_auth.py, and headers needs both http_auth.py and http_head.py. See also Dom's related blog entry.
